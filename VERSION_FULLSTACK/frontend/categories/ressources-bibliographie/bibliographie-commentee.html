Bibliographie commentée

Explorez une sélection de références essentielles sur les interfaces homme-machine, accompagnées de commentaires pour vous aider à choisir les lectures les plus adaptées à vos besoins.

"The Design of Everyday Things" – Don Norman

Commentaire : Un classique incontournable pour comprendre les principes fondamentaux du design. Don Norman explore comment rendre les interfaces intuitives en s’appuyant sur des exemples concrets, comme les portes mal conçues. Parfait pour les débutants comme pour les experts.

Type : Livre | Publié : 2013

"Don’t Make Me Think" – Steve Krug

Commentaire : Ce livre est une introduction accessible à l’utilisabilité. Steve Krug explique comment concevoir des interfaces qui ne font pas "penser" les utilisateurs, avec des conseils pratiques pour améliorer la navigation et la clarté. Idéal pour les designers UX.

Type : Livre | Publié : 2014

"10 Heuristics for User Interface Design" – Jakob Nielsen

Commentaire : Cet article propose 10 règles d’utilisabilité qui servent de guide pour évaluer et concevoir des interfaces. Très utile pour les audits d’interface ou pour comprendre les bases de l’ergonomie.

Type : Article | Publié : 1994

"Interaction Design: Beyond Human-Computer Interaction" – Preece, Rogers, Sharp

Commentaire : Un livre complet qui couvre les aspects théoriques et pratiques de la conception d’interactions. Idéal pour les étudiants ou professionnels cherchant une vue d’ensemble sur l’IHM moderne.

Type : Livre | Publié : 2019

"Designing for Accessibility" – Smashing Magazine

Commentaire : Cet article explore comment concevoir des interfaces accessibles, avec des exemples concrets et des conseils pour inclure tous les utilisateurs, notamment ceux en situation de handicap.

Type : Article | Publié : 2023
