Exercices de programmation

Développez vos compétences en programmation d’interfaces homme-machine avec ces exercices pratiques. Ces activités vous permettront de coder des IHM fonctionnelles tout en explorant des concepts comme l’interactivité, l’accessibilité et la réactivité.

Bouton interactif avec feedback

Créez un bouton HTML/CSS/JavaScript qui change de couleur au survol, affiche un message au clic, et respecte les standards d’accessibilité (ex. : focus clavier).

Niveau : Débutant | Langage : HTML/CSS/JS

Formulaire dynamique

Programmez un formulaire qui valide les champs en temps réel (ex. : email, mot de passe) et affiche des messages d’erreur sans recharger la page.

Niveau : Intermédiaire | Langage : JavaScript

Menu déroulant responsive

Développez un menu déroulant qui s’adapte aux écrans mobiles (hamburger menu) et aux écrans de bureau, avec des transitions fluides.

Niveau : Intermédiaire | Langage : HTML/CSS/JS

Tableau de bord interactif

Créez un tableau de bord avec des widgets interactifs (ex. : graphiques, notifications) en utilisant une bibliothèque comme React ou Vue.js.

Niveau : Avancé | Langage : React ou Vue.js

Interface vocale basique

Programmez une interface qui utilise l’API Web Speech pour répondre à des commandes vocales simples (ex. : "allume la lumière", "ouvre le menu").

Niveau : Avancé | Langage : JavaScript

Partagez vos solutions

Après avoir terminé un exercice, partagez votre code avec la communauté pour recevoir des retours et inspirer d’autres apprenants.
